The [Kwala Playground](https://kwala.network/dashboard) is your central hub for managing and testing workflows on the Kwala network. It provides an interactive environment where you can create, test, and manage your blockchain automation workflows.

## Key features

The Kwala Playground provides the following features:

- **My Workflows** - View and manage all your created workflows
- **KWALA Bounty** - Participate in bounty programs to earn rewards (claim up to 1 Sepolia ETH)
- **Workflow testing** - Test workflow configurations before deploying to production
- **Real-time debugging** - Debug action sequences and validate YAML syntax
- **Execution preview** - Preview how your workflows will execute

## Accessing the Playground

Navigate to [kwala.network/dashboard](https://kwala.network/dashboard) and connect your Metamask account to access the Playground.

## Dashboard sections

### KWALA Bounty

Participate in the Kwala Bounty program to:
- Complete tasks to earn rewards
- Claim Sepolia ETH for testing
- Learn the platform through hands-on challenges

### My Workflows

This is your workflow management hub where you can:

- View all workflows you've created
- Access workflow configurations
- Monitor workflow status
- Edit and update existing workflows
